# Zenith API

## PROJECT OVERVIEW
A robust and scalable API to manage product inventory. This API will serve as the backbone of a new online store, used by internal tools for inventory management and by the public-facing website to display products to customers.



## Prerequisites : Node.js , Javascript

## Installation
git clone https://github.com/uries242/zenith-api
cd [your-repo-name-here]
npm init -y
npm install express mongoose dotenv**
nodemon
